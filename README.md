# BinaryNet-on-tensorflow
binary weight neural network implementation on tensorflow

This is an implementation code for reproducing BNN

## How to run
```
python mnist.py
```

## Accuracy
| DataSet | accuracy |
|---------|----------|
| MNIST   |  99.04%  |
